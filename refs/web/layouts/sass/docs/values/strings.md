## –û–±–∑–æ—Ä

Sass –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ —Å—Ç—Ä–æ–∫, –≤–Ω—É—Ç—Ä–µ–Ω–Ω—è—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –∫–æ—Ç–æ—Ä—ã—Ö –æ–¥–∏–Ω–∞–∫–æ–≤–∞, –Ω–æ –∫–æ—Ç–æ—Ä—ã–µ –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –ø–æ-—Ä–∞–∑–Ω–æ–º—É: [—Å—Ç—Ä–æ–∫–∏ –≤ –∫–∞–≤—ã—á–∫–∞—Ö](#quoted), –Ω–∞–ø—Ä–∏–º–µ—Ä: `"Helvetica Neue"`, –∏ [—Å—Ç—Ä–æ–∫–∏ –±–µ–∑ –∫–∞–≤—ã—á–µ–∫](#unquoted) (—Ç–∞–∫–∂–µ –∏–∑–≤–µ—Å—Ç–Ω—ã–µ –∫–∞–∫ *–∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã*), –Ω–∞–ø—Ä–∏–º–µ—Ä, `bold`. –í–º–µ—Å—Ç–µ –æ–Ω–∏ –æ—Ö–≤–∞—Ç—ã–≤–∞—é—Ç —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Ç–∏–ø—ã —Ç–µ–∫—Å—Ç–∞, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ—è–≤–ª—è—é—Ç—Å—è –≤ CSS.

## –≠–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è

–í—Å–µ —Å—Ç—Ä–æ–∫–∏ Sass –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ CSS [–∫–æ–¥—ã —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è](https://developer.mozilla.org/en-US/docs/Web/CSS/string#Syntax):

* –õ—é–±–æ–π —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ –±—É–∫–≤—ã –æ—Ç A –¥–æ F –∏–ª–∏ —á–∏—Å–ª–∞ –æ—Ç 0 –¥–æ 9 (–¥–∞–∂–µ –Ω–æ–≤–∞—è —Å—Ç—Ä–æ–∫–∞!) –º–æ–∂–µ—Ç –±—ã—Ç—å –≤–∫–ª—é—á–µ–Ω –∫–∞–∫ —á–∞—Å—Ç—å —Å—Ç—Ä–æ–∫–∏, –Ω–∞–ø–∏—Å–∞–≤ –ø–µ—Ä–µ–¥ –Ω–∏–º `\`.
* –õ—é–±–æ–π —Å–∏–º–≤–æ–ª –º–æ–∂–µ—Ç –±—ã—Ç—å –≤–∫–ª—é—á–µ–Ω –∫–∞–∫ —á–∞—Å—Ç—å —Å—Ç—Ä–æ–∫–∏, –Ω–∞–ø–∏—Å–∞–≤ `\`, –∑–∞ –∫–æ—Ç–æ—Ä—ã–º —Å–ª–µ–¥—É–µ—Ç –µ–≥–æ [–Ω–æ–º–µ—Ä –∫–æ–¥–æ–≤–æ–π —Ç–æ—á–∫–∏ Unicode](https://en.wikipedia.org/wiki/List_of_Unicode_characters), –∑–∞–ø–∏—Å–∞–Ω–Ω—ã–π –≤ [—à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω–æ–º —Ñ–æ—Ä–º–∞—Ç–µ](https://en.wikipedia.org/wiki/Hexadecimal). –ü—Ä–∏ –∂–µ–ª–∞–Ω–∏–∏ –≤—ã –º–æ–∂–µ—Ç–µ –≤–∫–ª—é—á–∏—Ç—å –ø—Ä–æ–±–µ–ª –ø–æ—Å–ª–µ –Ω–æ–º–µ—Ä–∞ –∫–æ–¥–æ–≤–æ–π —Ç–æ—á–∫–∏, —á—Ç–æ–±—ã —É–∫–∞–∑–∞—Ç—å, –≥–¥–µ –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –Ω–æ–º–µ—Ä Unicode.

## –° –∫–∞–≤—ã—á–∫–∞–º–∏

–°—Ç—Ä–æ–∫–∏ –≤ –∫–∞–≤—ã—á–∫–∞—Ö –∑–∞–∫–ª—é—á–∞—é—Ç—Å—è –≤ –æ–¥–∏–Ω–∞—Ä–Ω—ã–µ –∏–ª–∏ –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏, –∫–∞–∫ –≤ `"Helvetica Neue"`. –û–Ω–∏ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å [–∏–Ω—Ç–µ—Ä–ø–æ–ª—è—Ü–∏—é](../interpolation/README.md), –∞ —Ç–∞–∫–∂–µ –ª—é–±–æ–π –Ω–µ—ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–∏–º–≤–æ–ª, –∫—Ä–æ–º–µ:

–°—Ç—Ä–æ–∫–∏ –≤ –∫–∞–≤—ã—á–∫–∞—Ö –≥–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ –±—É–¥—É—Ç —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω—ã –≤ —Å—Ç—Ä–æ–∫–∏ CSS, –∫–æ—Ç–æ—Ä—ã–µ –∏–º–µ—é—Ç —Ç–æ –∂–µ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ, —á—Ç–æ –∏ –∏—Å—Ö–æ–¥–Ω—ã–µ —Å—Ç—Ä–æ–∫–∏ Sass. –¢–æ—á–Ω—ã–π —Ñ–æ—Ä–º–∞—Ç –º–æ–∂–µ—Ç –≤–∞—Ä—å–∏—Ä–æ–≤–∞—Ç—å—Å—è –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –∏–ª–∏ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏ - —Å—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏, –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–∞ –≤ `"\""` –∏–ª–∏ `'"'`, –∞ —Å–∏–º–≤–æ–ª, –æ—Ç–ª–∏—á–Ω—ã–π –æ—Ç [ASCII](https://en.wikipedia.org/wiki/ASCII), –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–ª–∏ –Ω–µ –±—ã—Ç—å —Å–±–µ–∂–∞–ª. –ù–æ —ç—Ç–æ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –ø—Ä–æ–∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω–æ –æ–¥–∏–Ω–∞–∫–æ–≤–æ –≤ –ª—é–±–æ–π —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–π —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∞–º —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ CSS, –≤–∫–ª—é—á–∞—è –≤—Å–µ –±—Ä–∞—É–∑–µ—Ä—ã.
